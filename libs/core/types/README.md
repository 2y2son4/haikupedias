# @haikupedias/core/types

Core TypeScript type definitions for the Haikupedias project.

## Contents

- **Poetry types**: Word, HaikuLine, Haiku
- **Music types**: NoteValue, Bar, Composition
- **Haikupedia types**: Complete poem + music structure

## Usage

```typescript
import { Word, Composition, Haikupedia } from "@haikupedias/core/types";
```

This library contains only type definitions and interfaces. No runtime logic.
