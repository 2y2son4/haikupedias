# @haikupedias/core/utils

Core utility functions for the Haikupedias project.

## Contents

- **Constants**: Musical and structural constants (intervals, bar lengths, etc.)
- **Note utilities**: Pure functions for note arithmetic and transformations

## Usage

```typescript
import { addInterval, randomNote, MAJOR_THIRD } from "@haikupedias/core/utils";
```

All functions in this library are pure and have no side effects.
No Angular dependencies.
